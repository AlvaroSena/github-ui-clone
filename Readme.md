# GitHub UI Clone
Git repository of the GitHub Ui Clone Web App developed by Álvaro Sena

## Get Started
First clone the project and install all dependencies
````
git clone https://github.com/alvarosena/github-ui-clone.git
cd github-ui-clone
npm install
````
